Press Any Key to Start

Game Rules

1. Press any key to start the game.

2. Follow the pattern of colors as shown by the game.

3. Click the buttons in the same order as the game pattern.

4. Each correct sequence advances you to the next level.

5. If you make a mistake, the game will end, and you'll need to restart.

6. Good luck and have fun!
